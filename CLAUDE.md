## Project Overview

Master thesis project predicting how football players' performance qualities change after transfers, using **Twelve Football (Wyscout)** player/team data and **Transfermarkt** transfer records. The core model is a per-position Lasso regression: given a player's pre-transfer qualities and the tactical styles of origin/destination teams, predict post-transfer player qualities.

## Architecture

### Model versions (practice_model/)

- **v_1** (`practice_model/v_1/`): 8 leagues, ~7,400 transfers. Lasso wins (R²=0.369). XGBoost overfits.
- **v_2** (`practice_model/v_2/`): 228 competitions, ~78K rows. Lasso/Ridge still win (~R²≈0.31). XGBoost still overfits.

### Key concepts

**Twelve Quality Scores** (20 per player, from + to): Composite indices computed by Twelve Football from position-adjusted z-scores. These are the core player features for modelling:
> Active defence, Aerial threat, Box threat, Chance prevention, Composure, Defensive heading, Dribbling, Effectiveness, Finishing, Hold-up play, Intelligent defence, Involvement, Passing quality, Poaching, Pressing, Progression, Providing teammates, Run quality, Territorial dominance, Winning duels

**Team style qualities** (7 total): Defence, Defensive Transition, Attacking Transition, Attack, Penetration, Chance Creation, Outcome. Defined in `twelve_qualities/team_qualities.txt`.

**Positions** (5 outfield): Central Defender, Full Back, Midfielder, Winger, Striker.

**Train/test split**: Temporal — train on earlier seasons, test on most recent.

**Position-specific models**: Each position gets its own model. Some qualities have >50% nulls for certain positions and are excluded.
